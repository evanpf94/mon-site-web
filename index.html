Internet et Web

Le web est un service qui fonctionne grâce à Internet. Il permet de consulter des pages (HTML) reliées entre elles via des liens hypertextes.On y accede via un navigateur web (comme chrome ou Firefox). Le web n'est qu'une partie d'Internet, mais c'est celle que nous utilisons le plus souvent.
